2. Estruturas Fundamentais de Programação:

Tomando Decisões com Condicionais

O que são estruturas condicionais?

Para que um programa seja funcional, ele precisa tomar decisões. As estruturas condicionais permitem que seu código execute diferentes ações com base em critérios específicos.

Em Python, usamos if, elif e else para implementar essa lógica.

Entendendo a Lógica Condicional

Imagine um sistema de controle de acesso: se a senha estiver correta, acesso concedido; caso contrário, negado.

As condicionais guiam o fluxo de execução do programa, permitindo que ele se adapte a diferentes entradas.

A beleza das condicionais está na sua capacidade de criar programas dinâmicos e interativos.

Como funciona?

Estrutura básica

  • if (Se): Executa se a condição for verdadeira
  • elif (Senão, se): Verifica múltiplas condições
  • else (Senão): Executa se nenhuma condição anterior for verdadeira

A indentação é crucial em Python para definir blocos de código

Exemplo Prático 1: Cálculo de Reajuste Salarial

Cenário

A rede de farmácias "Descontão" precisa automatizar o reajuste salarial de seus colaboradores, com base em faixas salariais específicas.

Este é um problema clássico que demonstra a eficácia das estruturas condicionais.

Critérios de Reajuste

Faixa Salarial Percentual de Aumento
Até R$ 280,00 20%
Entre R$ 280,01 e R$ 700,00 15%
Entre R$ 700,01 e R$ 1500,00 10%
Acima de R$ 1500,00 5%

O programa deve solicitar o salário atual, calcular o novo salário e mostrar todas as informações relevantes.

Passo a passo para a solução

1. Obter o Salário: Usar input()

2. Converter o Valor: Converter string para float

3. Aplicar as Condições: Usar if, elif, else

4. Calcular o Aumento: aumento = (salário * percentual) / 100

5. Exibir o Resultado: Usar print() com f-strings

Código Python

# Cálculo de reajuste salarial
salario = float(input("Digite o salário do colaborador: R$ "))

if salario > 1500:
    percentual = 5
elif salario > 700:
    percentual = 10
elif salario > 280:
    percentual = 15
else:
    percentual = 20

aumento = (salario * percentual) / 100
novo_salario = salario + aumento

print(f"Salário antes do reajuste: R$ {salario:.2f}")
print(f"Percentual de aumento aplicado: {percentual}%")
print(f"Valor do aumento: R$ {aumento:.2f}")
print(f"Novo salário: R$ {novo_salario:.2f}")

Explicação do Código

salario = float(input(...)): Solicita e converte para float

if salario > 1500: percentual = 5: Verifica a primeira faixa

elif salario > 700: percentual = 10: Verifica a segunda faixa

elif salario > 280: percentual = 15: Verifica a terceira faixa

else: percentual = 20: Aplica para a última faixa

aumento = (salario * percentual) / 100: Calcula o valor do aumento

print(...): Exibe resultados formatados com 2 casas decimais

Exemplo Prático 2: Atribuição de Conceitos por Nota

Cenário

Um programa que lê duas notas parciais de um aluno, calcula a média e atribui um conceito e uma situação final (APROVADO ou REPROVADO).

Este exemplo demonstra como classificar dados com base em valores numéricos.

Tabela de Conceitos

Média de Aproveitamento Conceito
Entre 9.0 e 10.0 A
Entre 7.5 e 8.9 B
Entre 6.0 e 7.4 C
Entre 4.0 e 5.9 D
Entre 0.0 e 3.9 E

Passo a passo para a solução

1. Obter as Notas: Usar input() para duas notas

2. Calcular a Média: Converter para float e calcular (soma / 2)

3. Definir o Conceito: Usar if e elif com base na tabela

4. Verificar a Situação: if conceito in ["A", "B", "C"]: "APROVADO"

5. Exibir o Resultado: Usar print() com f-strings

Código Python

# Sistema de conceitos por média
nota1 = float(input("Primeira nota parcial: "))
nota2 = float(input("Segunda nota parcial: "))
media = (nota1 + nota2) / 2

if media >= 9.0:
    conceito = "A"
elif media >= 7.5:
    conceito = "B"
elif media >= 6.0:
    conceito = "C"
elif media >= 4.0:
    conceito = "D"
else:
    conceito = "E"

# Verificação simplificada de aprovação
resultado = "APROVADO" if conceito in ["A", "B", "C"] else "REPROVADO"

print(f"Nota 1: {nota1}")
print(f"Nota 2: {nota2}")
print(f"Média: {media:.2f}")
print(f"Conceito: {conceito}")
print(f"Resultado: {resultado}")

Explicação do Código

nota1, nota2 = float(input(...)): Solicita e converte para float

media = (nota1 + nota2) / 2: Calcula a média

if/elif/else: Determina o conceito baseado na média

resultado = "APROVADO" if conceito in ["A", "B", "C"] else "REPROVADO": Verificação simplificada

print(...): Exibe notas, média, conceito e resultado

3. Programação Modular com Funções

O que é programação modular?

É uma estratégia que divide um programa complexo em partes menores, mais simples e gerenciáveis, conhecidas como funções.

Pense em uma função como uma "mini-tarefa" ou "bloco de construção" que você define uma vez e reutiliza várias vezes.

Por que usar Funções?

Reutilização de Código: Evita repetição e torna o código mais eficiente

Organização e Legibilidade: Estrutura lógica e mais fácil de entender

Manutenção Simplificada: Alterações em um só lugar

Depuração Mais Fácil: Isola partes do código para identificar problemas

Definindo e Chamando Funções em Python

Estrutura básica

def nome_da_funcao(parametro1, parametro2):
    # Código da função
    return resultado

# Chamando a função
resultado = nome_da_funcao(valor1, valor2)

Funções são definidas com def, seguidas pelo nome, parâmetros entre parênteses e corpo indentado.

Exemplo: Somando Três Números

Cenário

Função simples para somar três números, ilustrando definição, argumentos e retorno.

Este exemplo demonstra como criar uma função reutilizável para operações matemáticas.

Passo a passo

1. Definir a Função: def somar(n1, n2, n3)

2. Realizar a Lógica: Converter para float e retornar soma

3. Obter Entrada: input() para três números

4. Chamar Função: soma_total = somar(num1, num2, num3)

5. Exibir Resultado: print()

Código Python

# Função para somar três números
def somar(n1, n2, n3):
    return float(n1) + float(n2) + float(n3)

num1 = input("Digite o primeiro número: ")
num2 = input("Digite o segundo número: ")
num3 = input("Digite o terceiro número: ")
soma_total = somar(num1, num2, num3)
print(f"A soma dos números é: {soma_total:.2f}")

Explicação do Código

def somar(...): Define função e retorna soma como float

inputs: Obtêm strings dos números

soma_total = somar(...): Chama função

print: Exibe soma formatada

4. Entrada e Saída de Dados (Input/Output)

Interagindo com o usuário

Para criar programas interativos, precisamos receber dados do usuário (input) e exibir resultados (output).

Em Python, as funções principais são input() e print().

Obtendo Dados com input()

A função input() solicita entrada do usuário via teclado

Sintaxe básica

variavel = input("Mensagem para o usuário: ")

O programa pausa até que o usuário digite algo e pressione Enter.

Importante: input() sempre retorna uma string

Ponto Crucial: input() retorna string!

Problema comum

Mesmo quando o usuário digita números, input() os retorna como texto (string).

Para realizar cálculos, é necessário converter para tipos numéricos (int ou float).

idade_str = input("Digite sua idade: ")
idade_int = int(idade_str)
idade_futura = idade_int + 5
print(f"Em 5 anos você terá {idade_futura} anos")

Exibindo Informações com print()

A função print() exibe informações no console

Formas de uso

print("Mensagem simples")
print(variavel)
print("Texto", variavel)
print(f"Texto com {variavel}")  # f-string

Recurso Avançado: f-strings

O que são f-strings?

Strings formatadas (f-strings) são uma forma moderna e eficiente de combinar texto e variáveis.

Elas são mais legíveis, concisas e flexíveis que outros métodos de formatação.

nome = "Maria"
idade = 25
print(f"Olá, {nome}! Você tem {idade} anos.")

Exemplos de f-strings

Exemplos práticos

# Formatação básica
nome = "Carlos"
idade = 30
print(f"Olá, {nome}! Você tem {idade} anos.")

# Cálculos dentro da f-string
largura = 10
altura = 5
print(f"Área: {largura * altura}")

# Formatação numérica
preco = 49.99
print(f"Preço: R$ {preco:.2f}")

5. Tipos de Dados e Manipulação

Tipos básicos em Python

Cada informação em Python tem um tipo de dado específico, que determina como ela é armazenada e quais operações podem ser realizadas.

int: Números inteiros (5, -100, 0)

float: Números decimais (3.14, -0.5, 10.0)

str: Textos ("Olá", "Python", "123")

bool: Valores lógicos (True, False)

A Importância da Manipulação

Conversão de tipos

Como vimos, input() sempre retorna strings. Para realizar operações matemáticas, precisamos converter esses valores para tipos numéricos.

Sem a conversão adequada, ocorrerão erros durante a execução do programa.

Erro comum: Tentar somar strings sem conversão

Convertendo Dados

Funções de conversão

  • float(): Converte para números decimais
  • int(): Converte para números inteiros
  • str(): Converte para texto
# Convertendo string para float
preco_str = "19.99"
preco_num = float(preco_str)

# Convertendo string para inteiro
idade_str = "25"
idade_num = int(idade_str)

Atenção: int() trunca decimais, não arredonda

Exemplo prático de conversão

Cálculo de aumento salarial

# Obtendo salário como string
salario_digitado = input("Digite seu salário: ")  # "1250.75"

# Convertendo para float
salario_numerico = float(salario_digitado)

# Calculando novo salário
novo_salario = salario_numerico * 1.10

# Exibindo resultado formatado
print(f"Seu novo salário é: R$ {novo_salario:.2f}")
